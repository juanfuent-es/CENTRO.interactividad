Dark Mode

Reduce la fatiga visual y aumenta la durabilidad de batería al reducir el consumo de energía en pantallas OLED

Explicación del Código

HTML: La Estructura

El HTML define la estructura básica de nuestra página. El elemento clave aquí es el <button>, que nos servirá como interruptor (toggle) para cambiar entre temas.


	<body>
		<h1>Dark Mode</h1>
		<button id="theme-toggle">Dark Mode</button>
	</body>

El atributo id="theme-toggle" es fundamental, ya que nos permite identificar y seleccionar este botón de forma única desde JavaScript.

CSS: Los Estilos y Temas

El CSS se encarga de la apariencia. Usamos variables CSS (declaradas en :root) para definir nuestras paletas de colores para los temas claro (default) y oscuro. El cambio de tema se logra simplemente añadiendo o quitando la clase .dark al <body>.


	/* Tema claro (default) */
	:root {
		--background-light: #f0f0f0;
		--text-light: #333;
		/* ... más variables */
	}
	
	body {
		background-color: var(--background-light);
		color: var(--text-light);
	}
	
	/* Tema oscuro */
	body.dark {
		background-color: var(--background-dark);
		color: var(--text-dark);
	}

JavaScript: La Funcionalidad

Aquí es donde ocurre la magia. JavaScript nos permite manipular el DOM (Document Object Model), que es la representación de nuestro HTML.

  1. Selección de Elementos del DOM: Primero, "capturamos" los elementos HTML con los que queremos interactuar. Usamos document.getElementById() para el botón y document.body para el cuerpo de la página.
  2. Event Listeners: Un "escuchador de eventos" (addEventListener) espera a que ocurra una acción específica en un elemento. En este caso, esperamos un evento 'click' sobre nuestro botón.
  3. Funciones: Una función es un bloque de código que realiza una tarea. Creamos una función que se ejecutará cada vez que el evento 'click' ocurra.
  4. Lógica del Toggle: Dentro de la función, usamos classList.toggle('dark'). Este método es muy útil: si la clase 'dark' no existe en el <body>, la añade; si ya existe, la elimina. Esto activa y desactiva los estilos que definimos en CSS.

	// 1. Selección de elementos del DOM
	const themeToggleButton = document.getElementById('toggle-theme-btn');
	const body = document.body;
	
	// 2. Añadir un Event Listener para el evento 'click'
	themeToggleButton.addEventListener('click', () => {
		// 3. Función que se ejecuta al hacer click
		// 4. Alterna la clase 'dark' en el elemento body
		body.classList.toggle('dark');
	
		// Opcional: Cambiar el texto del botón
		if (body.classList.contains('dark')) {
			themeToggleButton.textContent = 'Light Mode';
		} else {
			themeToggleButton.textContent = 'Dark Mode';
		}
	});
