Interactividad

Es la capacidad que puede tener un sistema, medios o dispositivos para responder a las acciones de un usuario.

Podríamos decir que la interactividad tecnológica es una conversación entre los usuarios y las máquinas, es la base de las experiencias digitales modernas, desde un click hasta instalaciones multimedia.

Interactividad Web

En un contexto web, esta comunicación se logra gracias a las respuestas del navegador a acciones del usuario.

Un usuario puede realizar acciones mediante dispositivos de entrada de datos: como mouse, cámaras, micrófonos, y disparando eventos como clicks, gestos, habla, eventos que generan datos, mismos que son procesados por el sistema y devuelve al usuario en forma de respuestas visuales o auditivas.

Podemos definir el proceso de interactividad de manera similar a la que se define un programa:

  1. Input: Datos de Entrada - Acciones del usuario
  2. Procesos - Procesamiento de datos de las acciones por el sistema
  3. Output: Salida - Retroalimentación visible/audible/sensible

Javascript

La base de la interactividad en la web

Javascript es un lenguaje de programación que nos permite que las interfaces web respondan a las interacciones de un usuario.
Se ejecuta sobre el navegador, no es necesario instalar ningún programa o aplicación externo o adicional. Es un lenguaje nativo en computadoras, el cual nos permite realizar múltiples procesos en la web, desde mensajes de alerta al usuario, validaciones en formularios, hasta animaciones en escenarios 3D.

JavaScript (JS) es el lenguaje que permite que las interfaces respondan. HTML estructura, CSS da estilo, JS da comportamiento.

¿Dónde vive JS?

  • Navegador: manipula la página, escucha eventos, guarda datos temporales.

¿Qué puede hacer en tu interfaz?

  • Cambiar contenido y atributos (mostrar/ocultar, actualizar textos e imágenes).
  • Validar formularios antes de enviarlos.
  • Reaccionar a gestos y teclas.
  • Pedir datos a servicios (APIs) y mostrarlos sin recargar la página.

Variables

Las variables son contenedores de datos que nos permiten almacenar y manipular información.

En JavaScript, las variables se declaran usando el operador =.

Ejemplo:

let nombre = "Juan";

En este ejemplo, la variable nombre almacena el valor "Juan".

Las variables pueden ser de diferentes tipos de datos:

  • Números: let edad = 25;
  • Cadenas de texto: let nombre = "Juan";
  • Booleanos: let esMayor = true;
  • Objetos: let persona = { nombre: "Juan", edad: 25 };
  • Arrays: let numeros = [1, 2, 3, 4, 5];

Las variables se pueden reasignar y modificar su valor.
